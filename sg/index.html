Style guide

Influenced by GOV.UK, PatternLab, MailChimp.

This is a work in progress.

Elements
Basic HTML elements, including: p, h1, h2, h3, h4, h5, ul, ol, table, blockquote, figure, dl, button, img.
Modules
Modules are "used to construct a more complex structure." Combinations of different elements used togather to create content groups.
Groups
Groups are collections of related modules.
